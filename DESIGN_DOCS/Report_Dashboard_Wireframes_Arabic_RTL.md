# Report Dashboard Wireframes - Arabic (RTL Layout)

## نظرة عامة (Overview)
تحويل تبويب التقارير إلى لوحة معلومات استخباراتية لمدير التدقيق مع دعم كامل للعربية وتكامل الملاحظات

## ملاحظات التنفيذ الخاصة بالعربية

### منطق ترتيب الأعمدة
```csharp
// الإنجليزية (LTR): الخطورة ← الملف ← النوع ← الفرع ← الملاحظات ← الحالة
// العربية (RTL): الحالة ← الملاحظات ← الفرع ← النوع ← الملف ← الخطورة
// الترتيب المنطقي يبقى نفسه، العرض المرئي ينعكس

DataGridColumn[] columns = isRtl
    ? new[] { statusCol, notesCol, branchCol, typeCol, fileCol, severityCol }
    : new[] { severityCol, fileCol, typeCol, branchCol, notesCol, statusCol };
```

### اختيار الخط
```xaml
<ResourceDictionary>
    <!-- English Font Family -->
    <FontFamily x:Key="PrimaryFont">Segoe UI, Calibri, Arial</FontFamily>

    <!-- Arabic Font Family -->
    <FontFamily x:Key="PrimaryFontAr">Arabic Typesetting, Tahoma, Traditional Arabic</FontFamily>

    <!-- Code must switch based on language setting -->
</ResourceDictionary>
```

### التباعد والتخطيط
```csharp
// العربية تتطلب ارتفاع سطر أكبر قليلاً للوضوح
double lineHeight = isArabic ? 1.5 : 1.2;
double letterSpacing = isArabic ? 0 : 0; // العربية لا تحتاج letter-spacing
```

### اتجاه FlowDirection في WPF
```xaml
<!-- Root container must set FlowDirection based on language -->
<Grid FlowDirection="{Binding CurrentFlowDirection}">
    <!-- All child elements inherit RTL automatically -->
</Grid>
```

```csharp
// In ViewModel or Code-behind
public FlowDirection CurrentFlowDirection =>
    ReportLocalizationService.IsArabic(_configStore)
        ? FlowDirection.RightToLeft
        : FlowDirection.LeftToRight;
```

## تحديات PDF العربية ومعالجتها

### 1. دعم RTL في PDF
```csharp
// iTextSharp example for RTL
PdfPTable table = new PdfPTable(6);
table.RunDirection = PdfWriter.RUN_DIRECTION_RTL; // Critical for Arabic

PdfPCell cell = new PdfPCell(new Phrase("النص العربي", arabicFont));
cell.RunDirection = PdfWriter.RUN_DIRECTION_RTL;
```

### 2. تضمين الخطوط العربية
```csharp
// Must embed Arabic fonts in PDF
BaseFont arabicFont = BaseFont.CreateFont(
    "C:\\Windows\\Fonts\\TAHOMA.TTF",  // Or Arabic Typesetting
    BaseFont.IDENTITY_H,  // For Unicode support
    BaseFont.EMBEDDED);   // Embed in PDF

Font font = new Font(arabicFont, 12);
```

### 3. محاذاة النص
```csharp
// Right-align all text in Arabic reports
Paragraph para = new Paragraph("الملخص التنفيذي", arabicFont);
para.Alignment = Element.ALIGN_RIGHT;
```

### 4. ترتيب الأعمدة في الجداول
```csharp
// Table columns must be reversed for Arabic
if (isArabic)
{
    table.AddCell(statusCell);
    table.AddCell(notesCell);
    table.AddCell(branchCell);
    // ... etc (reverse order)
}
else
{
    table.AddCell(severityCell);
    table.AddCell(fileCell);
    // ... etc (normal order)
}
```

- **قراءة الشاشة:** دعم JAWS وNVDA لقراء الشاشة العربية
- **تكبير الخط:** يجب أن يعمل 200٪ تكبير دون كسر التخطيط
- **الوضع عالي التباين:** ضمان عمل ألوان الخطورة في الوضع العربي عالي التباين
- **التنقل بلوحة المفاتيح:** يجب أن يتبع Tab ترتيب RTL المنطقي

---

## أهداف الأداء (نفس الإنجليزية)

- **التحميل الأولي:** < ثانيتين لـ 1,000 نتيجة
- **تصفية/فرز:** < 200 ميلي ثانية لـ 10,000 نتيجة
- **فتح لوحة الملاحظات:** < 100 ميلي ثانية للانزلاق وعرض الملاحظات
- **إنشاء التصدير:** < 5 ثوانٍ لـ PDF من 50 صفحة مع رسوم بيانية
