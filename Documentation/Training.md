# WorkAudit Training Guide

## Overview

Quick training outlines for different user roles.

---

## Training: Auditors

**Duration:** ~30 minutes

### Topics

1. **Login** – Username, password, roles.
2. **Workspace** – Browse documents, filters, status.
3. **Search** – Full-text search, filters.
4. **Reports** – Generate Branch Summary, Status Summary; view Report History.
5. **Assignments** – View My Assignments, Start Review, Mark Complete, View Document.

### Hands-On

- Log in and search for a document.
- Generate a Branch Summary report.
- Complete an assigned document (if assigned).

---

## Training: Managers

**Duration:** ~1 hour

### Topics

1. **All Auditor topics** (above).
2. **Assignments** – Assign documents, set due date, priority, notes.
3. **Assignment Management** – List, Calendar, Analytics; Reassign, Cancel, Bulk Assign.
4. **Drag-and-drop reassign** – Drag assignment row onto user in right panel.
5. **Reports** – Performance, Issues & Focus, Executive Summary; saved configs.
6. **Report Attestation** – Review and Approve reports.
7. **Control Panel** – KPI targets, scheduled reports, workflow settings.

### Hands-On

- Assign a document to a user.
- Reassign an assignment using drag-and-drop.
- Generate a Performance report and approve it.
- Configure a saved report configuration.

---

## Training: Administrators

**Duration:** ~1.5 hours

### Topics

1. **All Manager topics** (above).
2. **User Management** – Add, edit, reset password, roles.
3. **Control Panel** – Full configuration (General, Workflow, KPI, Backup, System).
4. **Scheduled Reports** – Enable, set type, time, save folder, email.
5. **SMTP** – Configure email for scheduled reports.
6. **Branches & Document Types** – Configure branches and document types.
7. **Audit Log** – Review and export audit trail.
8. **Backup & Restore** – Manual and automatic backup.

### Hands-On

- Add a new user.
- Configure scheduled reports with email.
- Run a backup and verify the backup file.
- Review the audit log for recent actions.

---

## Training: Executives

**Duration:** ~30 minutes

### Topics

1. **Executive Dashboard** – KPI cards, volume by branch, risk, trends.
2. **Executive Summary Report** – One-page PDF for management.
3. **Report History** – Access recent reports.
4. **Drill-down** – Double-click chart items to filter.

### Hands-On

- Open Executive Dashboard and review KPIs.
- Generate an Executive Summary report.
- Export dashboard to PDF.
