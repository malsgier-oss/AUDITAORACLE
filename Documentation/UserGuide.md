# WorkAudit User Guide

## Overview

WorkAudit is a bank-grade document audit and workflow management application. This guide helps end users perform daily tasks.

---

## Getting Started

### Logging In

1. Launch WorkAudit.
2. Enter your username and password.
3. Click **Login**.

### Main Interface

- **Activity Bar (left):** Navigate between Workspace, Import, Search, Reports, Assignments, Tools, and Archive.
- **Content Area:** Displays the selected view.
- **AI Chat (optional):** Toggle for natural language search and assistance.

---

## Workspace

- Browse documents by branch, section, and status.
- Use filters to narrow results.
- Double-click a document to open the preview.
- Select documents for bulk actions (tag, classify, assign).

---

## Import

- Import documents from folders or via webcam.
- Configure source path and options.
- Documents are classified and stored in the workspace.

---

## Search

- Full-text search across documents.
- Filter by branch, section, status, date range.
- Use AI Chat for natural language queries (e.g., "Find documents cleared last week").

---

## Reports

### Generating Reports

1. Go to **Reports**.
2. Select **Period** (Weekly, Monthly, 3 months, etc.).
3. Choose **Report Type** (Branch Summary, Performance, Executive Summary, etc.).
4. Apply filters (Branch, Section, Status) if needed.
5. Click **Generate PDF**, **Export Excel**, or **Export CSV**.

### Saved Configurations

- Click **Save configuration** to save your current filters and report type.
- Load a saved config from the dropdown for one-click report generation.
- Use the ★ button to mark favorites.
- **Manage saved configs** to edit, delete, or reorder.

### Report History

- Recent reports appear in the history panel.
- Use the filter box to search by type or date.
- Click an entry to open the report file (if it still exists).

---

## Assignments (My Assignments)

### Viewing Assignments

- Go to **Assignments** to see documents assigned to you.
- Filter by **Pending**, **In Progress**, **Completed**, or **All**.
- Check **Show overdue only** to focus on overdue items.

### Working on Assignments

1. Select an assignment.
2. Click **Start Review** to move it to In Progress.
3. Open the document with **View Document**.
4. When done, click **Mark Complete** (optionally add completion notes).

---

## Tools

- **Webcam:** Capture documents via camera.
- **Backup:** Create manual backups.
- **Control Panel:** (Admin only) Configure system settings.

---

## Archive

- View archived documents.
- Use retention and compliance filters.

---

## Tips

- Use **Ctrl+F** in search views for quick filtering.
- Report generation may take a few seconds for large datasets.
- Keep the app running if you use scheduled reports.
